# Iris Flower Classification 🌸

## CodeAlpha Data Science Internship — Task 1

### Overview
This project classifies Iris flowers into three species — **Setosa**, **Versicolor**, and **Virginica** — using measurements of sepal length, sepal width, petal length, and petal width.

### Models Used
| Model | Test Accuracy | CV Accuracy |
|-------|:---:|:---:|
| Logistic Regression | ✅ High | ✅ High |
| Decision Tree | ✅ High | ✅ High |
| Random Forest | ✅ High | ✅ High |
| SVM (RBF Kernel) | ✅ High | ✅ High |
| KNN (k=5) | ✅ High | ✅ High |

### Features
- Full Exploratory Data Analysis (Pairplot, Boxplots, Violin Plots, Heatmap)
- 5 ML models trained and compared
- 5-Fold Cross Validation
- Confusion Matrices for all models
- Best model selection with classification report

### Tech Stack
- Python, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn

### How to Run
```bash
pip install -r requirements.txt
python iris_classification.py
```

### Output
All visualizations are saved in the `figures/` directory.
